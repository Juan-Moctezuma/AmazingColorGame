Important Information

What is RGB?

This term stands for Red Green Blue, and refers to a model based on three color shades (hues of light) that can be combined in order to produce other color profiles. Each color's or channel's values can be adjusted; and these values range from 0 - 255. For instance, RGB(0,0,0) and RGB(255, 255, 255) are equivalent to pure black and all-white respectively.

FUN FACT

Combining red, green, and blue light is the standard method of producing color images on screens.



How does this game work?

The user needs to click on the color that matches the 'RGB(x,x,x)' being displayed up top. Please note that the displayed RGB colors change along with the square's colors. The goal is to estimate or guess the correct color. The user may select the level of difficulty (easy or hard).

Easy vs. Hard

The easy setting simply contains 6 squares, while the hard game has 9.



When is the game over?

A message displaying 'Try Again' shows up below the header to let the user know that the game is not done. Once the user selects the right color, 'correct' gets displayed.



How can the user reset any game?

Click on 'NEW COLORS' or 'PLAY AGAIN' (after winning). Switching between 'EASY' and 'HARD', or clicking on one of these resets the game as well.
